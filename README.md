# 🏠 House Price Prediction (Kaggle)

## 📌 Project Overview
This project aims to predict house prices using machine learning models based on the Kaggle House Prices dataset.

## ⚙️ Techniques Used
- Data Cleaning & Missing Value Imputation
- Feature Engineering
- Mutual Information for Feature Selection
- Log Transformation for Skewed Features
- Ensemble Learning:
  - Ridge Regression
  - Random Forest
  - Gradient Boosting
  - LightGBM
  - Stacking Regressor

## 📊 Model
Final model uses **Stacking Ensemble** combining multiple base models.

## 📈 Performance
- Kaggle Ranking: Top 0.12
